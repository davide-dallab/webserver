Aggiunta immagini, css e javascript

Per aggiungere immagini, è stato sufficiente supportare il Content-Type "image/*" e fare in modo di spedire il contenuto come uno stream di byte (il funzionamento è già dimostrato nella home page).

Il risultato è che se un'immagine viene inserita nella cartella docmuentRoot (come spiegato nel README.md) è possibile caricarla sulla pagina utilizzando il tag <img>. Per esempio, indicando nell'attributo src il valore "arena.jpg", sarà visualizzata la seguente immagine (si può anche cliccare per vederla singolarmente):

Con lo stesso metodo è stato possibile implementare il foglio di stile css che, come si vede in tutte le pagine, permette di apporre uno stile al contenuto HTML.

Infine è possibile collegare uno script utilizzando il metodo già descritto. Così facendo si può dichiarare un codice in un file a parte che può essere utilizzato su più pagine con semplicità. Segue un esempio che crea un pulsante arcobaleno.
